# 📊 E-Commerce Sales Analytics Dashboard

## 📌 Project Overview

This project presents an interactive **E-Commerce Sales Analytics Dashboard** built using **Power BI** to analyze sales performance, customer behavior, product categories, and profitability. The dashboard helps businesses make data-driven decisions by providing key performance indicators (KPIs) and interactive visualizations.

---

## 🎯 Objectives

- Analyze overall sales performance
- Monitor revenue, profit, and order trends
- Identify top-performing products and categories
- Understand customer purchasing behavior
- Compare sales across different states and time periods
- Support business decision-making through interactive dashboards

---

## 🛠️ Tech Stack

- **Power BI**
- **Microsoft Excel / CSV**
- **DAX**
- **Power Query**

---

## 📂 Dataset

The project uses two datasets:

- **Orders.csv** – Order details including sales transactions.
- **Details.csv** – Product and customer-related information.

---

## 📈 Dashboard Features

### Executive KPIs
- Total Sales
- Total Profit
- Total Orders
- Average Order Value (AOV)

### Sales Analysis
- Revenue by Month
- Profit Trend
- Sales Growth

### Product Analysis
- Top Selling Products
- Category-wise Sales
- Sub-category Performance

### Customer Analysis
- Customer Purchase Trends
- State-wise Sales Distribution
- High Value Customers

### Interactive Filters
- State
- Category
- Sub-Category
- Payment Mode
- Quarter

---

## 📊 Key Insights

- Identified the highest revenue-generating product categories.
- Analyzed monthly sales and profit trends.
- Detected high-performing states contributing maximum revenue.
- Compared payment methods based on sales performance.
- Highlighted profitable customer segments for business growth.

## 💡 Business Value

This dashboard enables businesses to:

- Track sales performance in real time.
